# Online Retail Data Analysis and Business Insights

## Project Overview

This project focuses on analyzing an online retail dataset to generate meaningful business insights. The analysis was performed using Python, SQL, and Power BI to evaluate sales trends, customer behavior, and product performance.

---

## Tools and Technologies

* Python (Pandas, NumPy)
* SQL (MySQL)
* Power BI
* Jupyter Notebook

---

## Key Tasks Performed

* Cleaned and preprocessed the dataset by handling missing values and correcting data types
* Removed invalid or inconsistent records to improve data quality
* Conducted exploratory data analysis (EDA) to identify trends and patterns
* Analyzed customer purchasing behavior and revenue distribution
* Built a dashboard to visualize key business metrics

---

## Key Insights

* Identified top-selling products and high-revenue categories
* Analyzed repeat customers and purchasing frequency
* Observed seasonal trends and peak sales periods
* Highlighted key contributors to overall revenue
